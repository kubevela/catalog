# VelaUX 介绍

VelaUX 提供端到端的应用交付和管理体验，其中包括了 API 服务和 UI 控制面板。

## 安装说明

1. 如果是生产使用，建议配置 MongoDB 数据库。
2. 配置 Domain 参数即可生成 Ingress 资源，否则只会生成 Service 资源。
